CSharpNewFeatures
=================

一个简单的项目，演示 C# 6.0 的语法新特性

CSharpNewFeatures 和 CSharpNewFeatures.Legacy 命名空间下的代码具有相同的功能，
前者使用了 C# 6.0 的新语法，后者只使用了 C# 5.0 的语法。

CSharpNewFeatures.Structs 命名空间下的代码演示了 C# 6.0 中针对结构体的语法增强，
没有对应的 C# 5.0 演示代码。

编译环境
--------

项目在 Visual Studio 2015 Preview 中编译通过。

运行环境
--------

项目运行在 .NET Framework 4.5 中，没有使用任何 .NET 4.6 及以上版本的新特性。
